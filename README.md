# mgm702 Homebrew Tap

Personal Homebrew tap for CLI tools and macOS applications by @mgm702.

## Usage

First, add the tap to your system:

```bash
brew tap mgm702/tap
```

Then install any available formula:

```bash
brew install <formula-name>
```

## Available Formulae

- **example-tool** — A description of your first CLI tool.

## Automation

This tap uses GitHub Actions to automatically update formulae when new releases are published in their respective source repositories.

## Maintenance

To update the tools you've installed:

```bash
brew update
brew upgrade
```
